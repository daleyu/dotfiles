# Utils

## branchy
You put in which branch you want and then it will allow you to select which o f

## diffy
Open up an FZF pane and then you can select the two files to diff with neovim.
So if I want to diff my local file with someone else's then I would probably end
up copy pasting their file into some temp file and then diffing it.

## goland
opens up goland

## gvm
go version manager, but I have replaced this with mise

## ij
opens up intellij

## ntemp
Opens up a temporary file with the output of a command.
- Typically used when running a service or something and I want to be able to
search and grep through the output logs

social blob transcriber. [deprecated no longer using social blob]
